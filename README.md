## 📌 Project Overview

This project is a responsive website developed for the **Institute of Digital Risk (IDR)**.
It showcases a modern UI with structured sections, smooth navigation, and clean design principles.

---

## 🚀 Features

* Responsive design (works on desktop & mobile)
* Smooth scrolling navigation
* Modern UI with custom fonts
* Structured layout using HTML, CSS, and JavaScript
* Live preview using VS Code Live Server

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Google Fonts

---

## 📁 Project Structure

```
assignment/
│── index.html
│── styles.css
│── script.js
│── idr-logo.svg
│── idr-icon.svg
│── package.json
│── package-lock.json

```

---

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/ananya489/idr-assignment.git
   ```

2. Open the folder:

   ```bash
   cd idr-assignment
   ```

3. Open `index.html` in browser
   OR use Live Server in VS Code

---

## 🌐 Live Preview

Run locally using Live Server:

* Right-click `index.html`
* Click **Open with Live Server**

---

## 📌 Notes

* `node_modules` is ignored using `.gitignore`
* Ensure dependencies are installed if using Node.js features
